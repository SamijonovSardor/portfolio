AI Form Builder

FormForge AI

Describe a form in plain language, get a complete, shareable form back in seconds.

Shipped
Source
Solo build Frontend + AI 11 technologies

An AI-powered form builder that turns a single natural-language prompt into a complete, validated, shareable form — built solo, end to end, on the modern React stack.

01AI Generation
Natural-language prompt sent to an LLM (OpenRouter / Cerebras, swappable via a single env var). Output validated against a runtime-generated Zod schema before it enters the builder.
02Custom DnD Engine
Drag-and-drop builder built from scratch with no external DnD library. Field reordering uses position-based transforms and CSS transitions for smooth, native-feeling drags.
03Live Preview & Public Page
Desktop/mobile preview toggle, plus a shareable public form page with client-side validation generated dynamically from the field schema.
04Appwrite Integration
Auth, TablesDB for forms & responses, Storage for file uploads — with public/private permissions scoped per table.
05Decoupled State
Builder, auth and UI state split across separate Zustand stores — fully decoupled from persistence, flushed to Appwrite only on save.
